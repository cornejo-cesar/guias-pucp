Pregunta guía para el evaluador

¿En qué medida el autor consiguió demostrar su tesis mediante una argumentación válida (lógica) y sólida (bien respaldada con evidencia)?

1 Secuencia lógica (argumentación válida)

1.1 ¿Qué significa?

Vínculo controversia (tema) - tesis (pregunta, postura, objetivos, hipótesis) - argumentos (razones) - evidencia (respaldos)

1.2 ¿Cómo se evalúa?

Estructura lógica - A nivel de todo el texto - Dentro de cada parte del ensayo - Dentro de cada párrafo

2 Uso de Evidencia (argumentación sólida)

2.1 ¿Qué significa?

  • Desarrollo suficiente de respaldos Demuestra un dominio de la literatura pertinente = La evidencia analizada es convincente para una audiencia de científicos sociales bien delimitada (la sociología política)
  • Dominio de la literatura
  • Originalidad

2.2 ¿Cómo se evalúa?

Intertextualidad (diálogo académico) = “desarrollo de ideas” Calidad el trabajo empírico: evidencia, datos y análisis - appropriateness - quantity - detail - accuracy
